# Campsite Monorepo

Campsite is a cozy static site generator with a friendly `create-campsite` scaffolder and a reusable `campsite-core` engine.

## Packages
- packages/create-campsite: `npm create campsite@latest` entrypoint
- packages/campsite-core: build, dev, and serve engine

## Develop
```
npm install
npm run dev
```
